# Contributing

Clone de repository, open its folder and install dependencies with:

```sh
npm i
```

Run it using:

```sh
npm start
```

## Dependencies

- Node.js
- Electron
- QRCode

## Features

- [x] Gerar qrcode
- [x] Salvar qrcode
- [x] Escolher o tamanho
- [x] Escolher cores
- [ ] Colocar logo
